# 🔐 Elliptic Curve Cryptography (ECC) Security System

## Overview

The **ECC Security System** is a command-line based cryptography project that demonstrates the implementation of **Elliptic Curve Cryptography (ECC)** using OpenSSL. The project provides an interactive menu-driven interface to perform secure cryptographic operations such as key generation, public key extraction, shared secret generation, digital signatures, hashing, encryption, and verification.

This project is designed for students, cybersecurity learners, and developers who want to understand the practical implementation of modern public-key cryptography.

---

## Features

* 🔑 Generate ECC Private Keys
* 🌐 Extract ECC Public Keys
* 🤝 Perform ECDH Shared Secret Generation
* 🔍 Compare Shared Secrets
* ✍️ Create Digital Signatures (ECDSA)
* ✅ Verify Digital Signatures
* 🔐 Encrypt and Decrypt Files
* #️⃣ Generate File Hashes
* 📄 Display Key Information
* 🗑️ Delete Generated Files
* 📂 Organize Cryptographic Files
* 🖥️ Interactive Menu-Driven Interface
* ⚡ Powered by OpenSSL

## Installation

### 1. Install OpenSSL

Download and install OpenSSL for Windows.

Verify installation:

```cmd
openssl version
```

---

### 2. Clone the Repository

```bash
git clone https://github.com/your-username/ecc-security-system.git
```

---

### 3. Open Project

Open Command Prompt inside the project folder.

Run:

```cmd
ecc.bat
```

or

```cmd
.\ecc.bat
```
